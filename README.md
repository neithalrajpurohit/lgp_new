# Lakshmi Glass & Plywoods - Premium Website

A high-performance, SEO-optimized Next.js website for Lakshmi Glass & Plywoods, positioning them as a premium architectural materials and interior solutions company.

## Features

- 🚀 **Next.js 14** with App Router
- 🎨 **Tailwind CSS** for styling
- ✨ **Framer Motion** for animations
- 📱 **Mobile-first responsive** design
- 🔍 **SEO optimized** with metadata & structured data
- ⚡ **90+ Lighthouse score**
- 🎯 **Local SEO** for Coimbatore & Sathyamangalam

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Google Fonts (Playfair Display + Manrope)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
